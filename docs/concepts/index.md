# Core Concepts

TempoEval is built around Focus Time and a three-layer evaluation stack.

## Framework at a glance
- Focus Time extraction (QFT, DFT, AFT)
- Layer 1: Retrieval metrics
- Layer 2: Generation metrics
- Layer 3: Reasoning metrics
- Composite: TempoScore
